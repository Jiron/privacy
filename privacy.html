4.9 DiscordDevTools API

Our website (https://www.discorddevtools.xyz/) offers a free API intended for developers. When using this API, in addition to technical data (e.g. IP address, browser details, and request timestamps), we may also collect and store manual user input such as text entries (e.g. names or messages), file inputs (e.g. images for generators), and URL parameters used in GET requests. For example, the welcome image generator (https://api.discorddevtools.xyz/welcome-image-generator) stores all parameters passed via the URL indefinitely, as they may be necessary for the operation or re-rendering of images or caching the images alltogether. These inputs are stored for analytical purposes and service continuity, but no personal data is processed unless explicitly submitted by the user via a request. If you wish to request deletion of data submitted to the API, please contact us at contact@jiron.dev with as much information as possible – ideally including the full GET request URL used during submission. We do not share API data with third parties, and data is only processed on infrastructure under our control.
